Exam Versions

Here is a sample of the sort of useful output you can make using the JSON generated by create-exam.py. The text here can easily be copy-pasta'd into a Word or Google document to be finalized by a non-programmer friend, doing things like adding page breaks when appropriate to make a print-ready exam version.

To view the raw JSON generated, navigate to localhost:5000/create-exam/sample?s=1337 where 1337 can be replaced by whatever seed value you want to use.
