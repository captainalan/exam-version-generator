Exam Version Generator

This web app creates multiple exam versions from user provided questions and answers. Check out an example.

Exam Versions for the 21st Century

It would be nice if students never cheated on exams. It would also be nice if ice cream was always good for you.

Fortunately, there are ways to prevent cheating and also situations where ice cream is indeed good for you (what would want to eat after being trapped on a desert island for a week withou food?!)

Features

  • Free, as in free beer and freedom!
  • Plain text formats make it easier to write exams in your favorite text editor
  • No red tape (though, this app may use "duct tape")
